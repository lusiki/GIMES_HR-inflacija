Eksploratorni pregled i konstrukcija indeksa medijske inflacije

Author

Published

23. prosinca 2025.

1 Uvod

Ovaj izvještaj predstavlja analizu medijske pokrivenosti inflacije u Hrvatskoj u razdoblju od siječnja 2021. do svibnja 2024. godine. Analiza se temelji na člancima prikupljenim iz hrvatskih online medija koristeći ključne riječi vezane uz inflaciju, cijene i troškove života.

1.1 Ciljevi analize

  1. Eksploratorni pregled - opis strukture i karakteristika prikupljenih podataka
  2. Vremenska analiza - dinamika medijskog izvještavanja o inflaciji kroz vrijeme
  3. Analiza izvora - koji mediji najviše pišu o inflaciji
  4. Konstrukcija indeksa - razvoj robusnog indeksa medijske inflacije (GIMES Inflation Index)

5 GIMES Indeks medijske inflacije

5.1 Metodologija

Indeks medijske inflacije (GIMES Inflation Index) konstruiran je kao kompozitni pokazatelj koji kombinira:

  1. Volumen - broj članaka o inflaciji (normalizirano)
  2. Intenzitet - prosječna duljina članaka (normalizirano)
  3. Sentiment - prosječni sentiment članaka (transformirano)
  4. Doseg - prosječni doseg članaka (normalizirano, ako dostupno)

Formula: \[GIMES\_Index = 0.4 \times Volume_{norm} + 0.2 \times Intensity_{norm} + 0.2 \times (1 - Sentiment_{norm}) + 0.2 \times Reach_{norm}\]

Indeks je skaliran na raspon 0-100, gdje viša vrijednost indicira veću medijsku pozornost na inflaciju.

Prikaži kod 
# ==============================================================================
# KONSTRUKCIJA GIMES INDEKSA
# ==============================================================================

# Dodaj ovo u kod nakon učitavanja podataka
if(!"AUTO_SENTIMENT" %in% names(dt)) {
  # Jednostavna heuristika za sentiment
  dt[, AUTO_SENTIMENT := fcase(
    stri_detect_regex(tolower(FULL_TEXT), 
                     "kriza|katastrofa|strah|panika|pogoršanj|problem", 
                     case_insensitive = TRUE), -1,
    stri_detect_regex(tolower(FULL_TEXT), 
                     "pad inflacije|usporavanje|stabilizacij|poboljšanj", 
                     case_insensitive = TRUE), 1,
    default = 0
  )]
}


# Mjesečna agregacija svih komponenti
if("AUTO_SENTIMENT" %in% names(dt)) {
  index_data <- dt[, .(
    volume = .N,
    avg_length = mean(text_length, na.rm = TRUE),
    avg_sentiment = mean(AUTO_SENTIMENT, na.rm = TRUE),
    avg_reach = if("REACH" %in% names(.SD)) mean(REACH, na.rm = TRUE) else NA_real_,
    total_reach = if("REACH" %in% names(.SD)) sum(REACH, na.rm = TRUE) else NA_real_,
    avg_interactions = if("INTERACTIONS" %in% names(.SD)) mean(INTERACTIONS, na.rm = TRUE) else NA_real_
  ), by = yearmonth][order(yearmonth)]
} else {
  # Ako nema AUTO_SENTIMENT, koristi neutralnu vrijednost
  index_data <- dt[, .(
    volume = .N,
    avg_length = mean(text_length, na.rm = TRUE),
    avg_sentiment = 0,  # Neutralna vrijednost
    avg_reach = if("REACH" %in% names(.SD)) mean(REACH, na.rm = TRUE) else NA_real_,
    total_reach = if("REACH" %in% names(.SD)) sum(REACH, na.rm = TRUE) else NA_real_,
    avg_interactions = if("INTERACTIONS" %in% names(.SD)) mean(INTERACTIONS, na.rm = TRUE) else NA_real_
  ), by = yearmonth][order(yearmonth)]
}


# Normalizacija funkcija (min-max scaling na 0-100)
normalize <- function(x) {
  if(all(is.na(x))) return(rep(50, length(x)))
  min_x <- min(x, na.rm = TRUE)
  max_x <- max(x, na.rm = TRUE)
  if(max_x == min_x) return(rep(50, length(x)))
  (x - min_x) / (max_x - min_x) * 100
}

# Normalizacija komponenti
index_data[, `:=`(
  volume_norm = normalize(volume),
  length_norm = normalize(avg_length),
  sentiment_norm = normalize(avg_sentiment),  # Viši sentiment = niži indeks
  reach_norm = normalize(avg_reach)
)]

# Izračun GIMES indeksa
# Negativni sentiment povećava indeks (više negativnih vijesti = veća percepcija inflacije)
index_data[, gimes_index := 
  0.40 * volume_norm + 
  0.20 * length_norm + 
  0.20 * (100 - sentiment_norm) +  # Invertiran sentiment
  0.20 * fifelse(is.na(reach_norm), 50, reach_norm)
]

# Klizni prosjek za glađenje
index_data[, gimes_index_ma3 := frollmean(gimes_index, n = 3, align = "center")]

# Godišnja promjena (YoY)
index_data[, gimes_yoy := (gimes_index / shift(gimes_index, 12) - 1) * 100]

7.2 Interpretacija

Na temelju provedene analize možemo zaključiti sljedeće:

  1. Medijska pozornost prati ekonomsku realnost - GIMES indeks pokazuje jasnu korelaciju sa službenim podacima o inflaciji, što sugerira da mediji adekvatno prate inflacijske pritiske.

  2. Vršne vrijednosti - Najveća medijska pozornost na inflaciju zabilježena je u razdoblju najviše službene inflacije, što potvrđuje reaktivnu prirodu medijskog izvještavanja.

  3. Sentiment - Prosječni sentiment članaka o inflaciji je pretežno negativan, što je očekivano s obzirom na prirodu teme.

  4. Regionalna pokrivenost - Nacionalni i poslovni mediji dominiraju u izvještavanju, dok regionalni mediji imaju manji, ali značajan udio.
